# Money-Master - Finance Tracking CLI Application

Money-Master is a command-line-based finance tracking application written in Python. It's designed to help you manage your finances with precision, offering the flexibility to track and manage expenses based on the 50/30/20 rule or any custom ratio you prefer.

## Features

- **Expense Tracking**: Easily record and categorize your expenses.
- **Income Management**: Keep track of your income sources.
- **Budgeting**: Create and manage budgets based on the 50/30/20 rule or customize your own ratios.
- **Financial Insights**: Gain valuable insights into your financial health with summary reports.
- **SQLite Database**: Securely store your financial data using SQLite for data integrity.

Follow the on-screen instructions to track your expenses, manage income, and create budgets.

## Configuration

You can customize the expense ratio by modifying the config.py file. The default expense ratio is set to the 50/30/20 rule (50% for needs, 30% for wants, 20% for savings), but you can adjust this to match your specific financial goals).
